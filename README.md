# android-mobile-app-test

Implemented Android mobile automation test framework using Appium with Page Object Model (POM) architecture, including Driver Factory, reusable utilities, Base Test setup, page classes, and scalable test execution structure. Utilized Java, Appium, Selenium WebDriver, TestNG/NUnit, Maven/Gradle, Android Studio, ADB, UiAutomator2, Appium Inspector, Git, Jenkins/Azure DevOps CI/CD, and reporting/logging tools for maintainable and efficient mobile test automation.
